A Visual Message for the Inhabitants of Other Celestial Bodies

The idea for this message comes from Stanley Kubrick's film 2001: A Space Odyssey. The first scene depicts early humans learning to use tools to kill each other. This is one of the main things that distinguishes us from any other species we have ever come across - we actively kill each other and always have. That being said, I don't think things have to be that way and I think most people would agree. This is why the last person is extending upwards towards space above him. It is almost like a cry for help to whoever sees the message to help us before we kill ourselves to extinction.
